# What's not working  💻
Type | Info | Status
:---------|:---------|:----------
Brightness control | Brightness level is coming from Windows. If you set your brightness %50 in Windows. Your macOS level is %50 | ⚠️
HDMI | Beacuse it connected to RTX2060 | ❌
Airdrop, Sidecar | Beacuse Intel Wi-Fi Doesn't Support | ❌

# What's you have to do  💻
Type | Info | Status
:---------|:---------|:----------
SMBIOS Settings  | ⚠️ 
Rename config    | ⚠️ 
 
- With OpenCore Configurator you should definitely set your SMBIOS settings because the config does not contain SMBIOS information MacBook Pro 16.4
